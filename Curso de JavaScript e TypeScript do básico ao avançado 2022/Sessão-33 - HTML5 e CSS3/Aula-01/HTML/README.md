**<h1> ao <h6>** = São utilizados para marcar a importância dos títulos, sendo <h1> o mais importante e <h6> o menos. Uma dica: use apenas um <h1> por página, pois ele representa o objetivo da sua página.

**<p> **representa um parágrafo, mas ele não suporta apenas texto, podemos adicionar imagens, código, vídeos e vários outros tipos de conteúdo dentro dele.

**<'br>** = Pula uma linha. 

<'a href=""> </a> = que significa anchor/âncora, ele representa um hyperlink, é ele que interliga vários conteúdos e páginas na web. O href representa o *hyperlink* para onde sua âncora aponta, pode ser uma página do seu ou de outro site, um e-mail e até mesmo um telefone, os dois últimos precisam dos prefixos mailto: e tel:, respectivamente.

<b> ou <bold> = Utilizado para deixar o texto negrito. / <strong>  = utilizado para da o texto uma forte importância, como um chamado de atenção.  

<i> = Itálico. 

<em> = Da ênfase ao texto. 

<img src="" alt="" width="" height="">  =  No "Src" é aonde a imagem está, pode ser adicionado o link. / "Alt" recebe o nome da imagem. / "Width" recebe a largura. / "Height" recebe a altura. 

<del> = Texto excluído, riscado no meio. Tag semântica. / <u> = Texto excluído, riscado no meio. Tag física.

<ins> = Texto inserido, riscado embaixo. Tag semântica. / <s> = Texto inserido, riscado embaixo. Tag física.

<small> = Deixa o texto menor. 

**<sup>** = Utilizado pra elevar um valor, ex: 2<sup>2</sup> // <sub> = Pra descer um valor H<sub>2</sub>0

**<q>** = Citar um texto, ex: <q cite="https://pt.wikipedia.org/wiki/Texto">Texto</q>  ou <'blockquote> = Bloco de citação.

<'hr> =  representa uma quebra temática entre elementos de nível de parágrafo (por exemplo , uma mudança da cena de uma história, ou uma mudança de tema com uma seção).

**<pre><code>** = Adiciona um código ao HTML

<span> = é um contêiner genérico em linha para conteúdo fraseado, física. / <div> é um container genérico para conteúdo de fluxo, semântica. 

<'ul> = Representa uma lista de itens sem ordem rígida Ex: <'ul><a href="#">  ou <'ol> = Representa uma lista de itens ordenada por números. Ex: <'ol><li> / <li> = É usado para representar um item que faz parte de uma lista.
